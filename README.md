# MustUI-Studio

**MustUI-Studio** is a development environment where the components of the `must-ui` React UI library are showcased, tested, and documented.

This project is not a UI library.  
The actual library is the **must-ui** npm package.

---

## Purpose

- Showcase `must-ui` components in a live environment
- Test component variants and props
- Preview Light / Dark mode appearances
- Provide fast preview and documentation during development

---

## Technologies Used

- **Next.js**
- **TypeScript**
- **Tailwind CSS**
- **must-ui**
- Light / Dark mode support
